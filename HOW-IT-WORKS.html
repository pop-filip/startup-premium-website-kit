Jedan brief fajl. Jedna komanda. Kompletan website — od koda do servera.

01 · Ideja

Šta je ovo?

Osobni starter sistem koji ti omogućava da pokreneš profesionalan website za bilo koji biznis — brzo, konzistentno i bez ponavljanja posla od nule svaki put.

📋
BRIEF.md
Popuniš jedan template fajl sa svim informacijama o projektu — naziv, domena, sadržaj, boje, slike, kontakt, legal. Claude sve ostalo radi automatski.
⚡
Init Script
Jedan bash script kreira GitHub repo, kopira strukturu iz kita, generiše sve fajlove i pushuje. Projekt je spreman za razvoj za manje od minute.
🚀
Auto Deploy
GitHub Actions — svaki git push main automatski deployhuje na Hetzner VPS. Nema ručnog uploadanja.
✅
Checklist
Interaktivni checklist prati šta je urađeno — SEO, security, analytics, legal, testing. Svaki projekat je kompletno pokriven.
02 · Workflow

Kako radi — korak po korak

Od prve ideje do live sajta — ovaj je proces.

1
Pripremi BRIEF.md
Kopiraj BRIEF_TEMPLATE.md iz kita, preimenuj ga u naziv projekta i popuni. Ne trebaš sve sekcije — ostavi prazno ono što ne trebaš. Minimalni brief: naziv, domena, jezik, hero tekst, kontakt.
10–20 min BRIEF_TEMPLATE.md
2
Reci Claudeu
Otvori novi Claude Code razgovor i reci mu gdje je brief. Claude čita CLAUDE.md iz kita i zna tačno šta da radi — bez dugih objašnjenja svaki put iznova.
new-website, brief je u ~/projekti/salon-bella/brief.md
automatski CLAUDE.md
3
Init script se pokreće
Claude pokreće init-website.sh koji čita brief i kreira kompletan projekt: GitHub repo, sve fajlove, nginx config, GitHub Actions workflow.
bash /Users/filip/tool-kit/init-website.sh --brief brief.md
~30 sekundi GitHub repo nginx config
4
Claude gradi HTML
Claude čita brief i gradi kompletan HTML fajl sekciju po sekciju — sa svim pravilima kita: WebP slike, SVG bez style bloka, Schema.org, GA4 sa cookie consent, security headers, responsive dizajn.
automatski iz brief-a SEO ready iOS Safari safe GDPR compliant
5
Server setup (jednom po domeni)
SSH na jebenko server, pokreni setup script sa domenom. On kreira web direktorij, aktivira nginx config i izdaje SSL certifikat automatski.
ssh root@157.180.67.68
bash setup-vps.sh bella-linz.at salon-bella
jednom po projektu SSL automatski
6
GitHub Secrets → Auto Deploy aktivan
Dodaj 4 GitHub Secrets u repo settings. Od tog trenutka — svaki git push main automatski deployhuje na server. Nema ručnog rada.
SSH_PRIVATE_KEY SSH_HOST SSH_USER DEPLOY_PATH
7
Go Live + Post-Launch
DNS usmjeri na server IP. Provjeri checklist.html — Google Search Console, GA4 real-time, UptimeRobot, OG preview. Sajt je live i monitorizovan.
live 🎉 Search Console UptimeRobot

05 · Pravila kita

Što Claude uvijek primjenjuje

Svaki generisani sajt automatski poštuje ova pravila — nema potrebe da ih ponavljaš.

01
SVG kao <img> — nikad <style> blok unutar SVG fajla. iOS Safari ne renderuje CSS klase u SVG-u učitanom kao slika. Uvijek inline fill atributi.
02
WebP slike — uvijek <picture> element sa PNG fallbackom. iOS < 14 ne podržava WebP. Bez fallbacka = prazni kvadratići na starim iPhonima.
03
GA4 se učitava tek nakon cookie consent-a. GDPR zahtijeva eksplicitni pristanak. Analytics skripta se injektuje u DOM tek kad korisnik prihvati.
04
Schema.org na svakoj stranici. Organization + tip biznisa iz brief-a (LocalBusiness, HairSalon itd.) · LocalBusiness sa adresom i radnim vremenom.
05
Security headers u nginx.conf. X-Frame-Options, X-Content-Type-Options, Referrer-Policy, HSTS, Permissions-Policy — automatski u svakom config-u.
06
Cache-Control: slike 30d, HTML no-cache. Slike i fontovi se agresivno keširaju, HTML nikad — znači promjene su vidljive odmah.
07
Mobile-first, testirati na iOS Safari. Sve animacije, CubicBezier vrijednosti, filter efekti moraju raditi na iPhoneu.
08
Impressum je obavezan za .at i .de domene. Zakonska obaveza — generiše se automatski iz brief-a ako je domena .at ili .de.
09
Sve slike imaju width, height i alt atribut. Sprječava CLS (layout shift) i poboljšava SEO i accessibility.
10
JavaScript defer ili async — nikad u <head> bez atributa. Blokira rendering. Sve skripte idu na kraj <body> ili sa defer atributom.
